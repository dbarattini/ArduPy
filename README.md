## About

Framework that allows you to easily control arduino with raspberry

## Where to start

### Requirements

- python 3.7.3
- pyserial 3.4

install requirements:

```bash
cd ardupy
pip install -r requirements.txt
```

if you have all the requirements installed, you can start using ardupy.
